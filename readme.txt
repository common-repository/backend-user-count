Editors can now view your website's current user count!

== Description ==

<p>This simple plugin will allow all editors of a WordPress website to view the current user count.</p>
<p>The plugin will show you the total count of users as well as counts-per-role.</p>

== Installation ==

<p>All you have to do is upload the plugin within your WordPress Plugins menu and activate it.</p>
<p>Then, just login as an editor or administrator and visit the menu page that appears to see your current user view counts.</p>

== Frequently Asked Questions ==

= What can this plugin do? =

This simple plugin is for administrators that want to give their editors access to their website's user counts.
